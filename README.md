# 工具说明
使用AiPy工具编写的批量文件重命名工具，python编写打包为exe。

# 使用说明
打开exe，选择要批量重命名文件的目录，填写重命名后的前缀或后缀，选择重命名文件的排序方式
<img width="810" height="760" alt="image" src="https://github.com/user-attachments/assets/a4b94df8-775e-4137-a47f-872895847c49" />

选择命名的数字样式（阿拉伯数字、罗马数字、汉字等）点击刷新预览，预览修改后的名字样式
<img width="801" height="734" alt="image" src="https://github.com/user-attachments/assets/8d72feec-0b59-46f4-bab7-5fc6a57ad687" />

根据预览调整样式，执行重命名即可
